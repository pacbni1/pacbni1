    Conceptos importantes de este módulo son: HTML Es un lenguaje de etiquetado o de marcado que se utiliza para el desarrollo y confección de páginas web en Internet.Son las siglas de HyperText Markup Language o Lenguaje de Marcado de Hipertexto.Está basado en SGML. SGML Son las siglas de Standard Generalized Markup Language o lenguaje de marcado generalizado estándar. Es el estándar ISO 8879 del año 1986 que establece lenguajes de marcado para documentos de forma generalizada de forma declarativa y rigurosa. XHTML Son las siglas de eXtensible HyperText Markup Language o lenguaje extensible de marcado de hipertexto.Establece una sintaxis y unas normas rigurosas para el código HTML para que este sea expresado como XML válido. XML Son las siglas de eXtensible Markup Language o lenguaje de marcado extensible o lenguaje de marcas extensible.Es un meta-lenguaje desarrollado por W3C que permite crear y definir otros lenguajes de marcas.
